# הוראות התקנה של אפליקציית ג'מיני אולטרה

שלום וברוכים הבאים לאפליקציית ג'מיני אולטרה. במדריך זה תמצאו הוראות התקנה מפורטות כדי להתחיל לשימוש באפליקציה.

## דרישות מערכת
- מערכת הפעלה: Windows, macOS, או Linux
- גרסה עדכנית של Node.js (גרסה 14 ומעלה)
- npm (מנהל חבילות Node)

## שלב 1: הורדת הקוד מקור

1. פתחו את מסך הטרמינל במחשב שלכם.
2. המריצו את הפקודה הבאה כדי להוריד את הקוד ממאגר GitHub:
   ```bash
   git clone https://github.com/asafnovlevy-bot/gemini-ultra-app.git
   ```

## שלב 2: התקנת תלויות

1. עברו לקטנועה שהורדתם:
   ```bash
   cd gemini-ultra-app
   ```
2. התקינו את התלויות הנדרשות באמצעות npm:
   ```bash
   npm install
   ```

## שלב 3: הרצת האפליקציה

כדי להריץ את האפליקציה, השתמשו בפקודת:
```bash
npm start
```

היישום יתחיל לרוץ על http://localhost:3000.

בהצלחה!
